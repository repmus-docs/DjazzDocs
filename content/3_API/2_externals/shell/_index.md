Shell is a Max object that allows Max to interact with the Bash shell on Mac OS. It is [available here](https://github.com/jeremybernstein/shell/releases/tag/1.0b3). 

It is used in the "make_score_file.maxpat" tool; it opens a shell instance in order to call the {{< apipage "2_externals/midifile" >}}midifile external{{< /apipage >}}.
